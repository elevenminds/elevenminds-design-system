Iconography

Naanix layer over the elevenminds icon set

NAANIX uses the full elevenminds icon library but in solid circular tiles rather than the outline roundels — denser, more affirmative, sized 40–48px in the toolbar. Red tiles signal destructive or alert actions; slate tiles for navigation; outlined tiles for secondary tools.

Voice

Spanish-first · operator tone

Direct, in present tense

"Unidad T-454 fuera de ruta a las 11:38." Not "We've detected that unit T-454 may have left its assigned route."

Operator vocabulary

Flotilla, recorrido, parada, geocerca, posta, telefonema. Avoid translated SaaS-isms (no "engagement", no "dashboard").

Alerts use the ladder, not adjectives

Color carries severity. Copy stays factual: "Velocidad 86 km/h. Límite 80." No "uh-oh", no "we noticed".
